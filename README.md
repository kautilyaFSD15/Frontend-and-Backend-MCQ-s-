## 🔥 Hardest Questions (Can You Get These?)

<details>
<summary>🧠 Challenge #1 — JavaScript</summary>

<br/>

```javascript
console.log(1 + "2" + 3)
```

**A)** `6` &nbsp; **B)** `"123"` &nbsp; **C)** `33` &nbsp; **D)** `Error`

<details><summary>👁️ Reveal Answer</summary>

✅ **B — `"123"`**

`1 + "2"` → `"12"` (number gets coerced to string)
`"12" + 3` → `"123"` (3 gets coerced to string)

</details>
</details>

<details>
<summary>🧠 Challenge #2 — JavaScript</summary>

<br/>

```javascript
console.log([] == false)
```

**A)** `true` &nbsp; **B)** `false` &nbsp; **C)** `TypeError` &nbsp; **D)** `undefined`

<details><summary>👁️ Reveal Answer</summary>

✅ **A — `true`**

`[]` → coerced to `""` → coerced to `0`
`false` → coerced to `0`
`0 == 0` → `true` 🤯

</details>
</details>

<details>
<summary>🧠 Challenge #3 — CSS</summary>

<br/>

Which element has **higher specificity**?

```css
/* A */
#header .nav a { color: red; }

/* B */
.header .nav .link { color: blue; }
```

<details><summary>👁️ Reveal Answer</summary>

✅ **A wins** — specificity `(0,1,1,1)` vs `(0,0,3,0)`

IDs beat any number of classes. One ID > a hundred classes.

</details>
</details>

<details>
<summary>🧠 Challenge #4 — Node.js</summary>

<br/>

What is the **execution order**?

```javascript
console.log("1");
setTimeout(() => console.log("2"), 0);
process.nextTick(() => console.log("3"));
console.log("4");
```

<details><summary>👁️ Reveal Answer</summary>

✅ **Output: `1 → 4 → 3 → 2`**

- Sync runs first: `1`, `4`
- `process.nextTick` runs before event loop: `3`
- `setTimeout` runs in macrotask queue: `2`

</details>
</details>
