Q1. Node.js is:
A. Programming language
B. JavaScript runtime environment
C. Database
D. Framework
✅ Answer: B
Explanation: Node.js runs JavaScript outside the browser.

Q2. Node.js is built on:
A. SpiderMonkey
B. V8 Engine
C. Chakra
D. Java VM
✅ Answer: B

Q3. Which module is used for file handling?
A. http
B. fs
C. path
D. os
✅ Answer: B

Q4. Which module is used to create server?
A. fs
B. os
C. http
D. url
✅ Answer: C

Q5. What does NPM stand for?
A. Node Package Manager
B. New Package Manager
C. Network Package Manager
D. Node Project Manager

Q6. require() is used for:
A. Exporting modules
B. Importing modules

Q7. Node.js is:
