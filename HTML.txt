Q1. HTML stands for:
A. Hyper Trainer Marking Language
B. Hyper Text Markup Language
C. Hyper Text Marketing Language
D. Hyper Tool Multi Language
✅ Answer: B
Explanation: HTML is the standard markup language for web pages.

Q2. Which tag defines the largest heading?
A. <h6>
B. <heading>
C. <h1>
D. <head>
✅ Answer: C
Explanation: <h1> is the highest-level heading.

Q3. Which is a self-closing tag?
A. <div>
B. <p>
C. <img>
D. <section>
✅ Answer: C
Explanation: <img> is a void element (no closing tag).

Q4. Which tag is used to create a hyperlink?
A. <link>
B. <a>
C. <href>
D. <url>
✅ Answer: B
Explanation: <a> tag creates hyperlinks.

Q5. Which attribute specifies image path?
A. href
B. src
C. alt
D. path
✅ Answer: B

Q6. Which is a semantic tag?
A. <div>
B. <span>
C. <section>
D. <b>
✅ Answer: C
Explanation: Semantic tags describe meaning.
